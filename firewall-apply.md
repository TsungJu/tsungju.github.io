# Firewall Apply System

## What is it ?

此專案為部署於Heroku的防火牆開放申請系統，目前提供的功能如下：

 - 防火牆開放申請。
 - 查看目前所有正在處理中的防火牆開放申請。

## How to use ?

1. home頁顯示目前所有正在處理中的防火牆開放申請：

![Home Page](site_static/home_page.JPG)

2. 填寫相關防火牆開放，並送出申請：
